# guide-china-yibosports

本仓库用于归档和发布多个独立的 HTML 页面。内容以静态页面形式存放，不针对任何特定网站或域名。所有页面均为历史资料或独立说明文档，仅供存档与参考。

## 项目简介

这是一个以 HTML 页面归档为主要目标的仓库。页面内容独立、结构清晰，无需依赖外部数据库或后端服务。仓库本身不提供任何形式的营销、推广或跳转服务。

## 目录说明

```
.
├── index.html          # 仓库首页/索引页
├── pages/              # 存放所有独立 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 静态资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `index.html`：仓库的入口页面，通常包含页面列表或导航。
- `pages/`：所有独立 HTML 页面存放于此，按需命名。
- `assets/`：页面所需的公共资源文件。

## 页面归档说明

- 每个页面均为完整的独立 HTML 文件，可直接在浏览器中打开。
- 页面内容可能包含说明性文字、数据或参考信息，不包含交互式商业功能。
- 归档内容不保证实时更新，请以实际文件为准。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/guide-china-yibosports.git
   ```
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意页面。
3. 也可通过 GitHub Pages 等静态托管服务部署后访问。
